# Flutter-Cache-Tests

Here there are some demo created to test various implementation of cache into Flutter Mobile Apps 

1. [bloc_hive_caching_data](<https://github.com/DavideNas/DEMO-Flutter-Cache-Tests/tree/main/bloc_hive_caching_data>) : Demo with Bloc Pattern & Hive Caching
2. [cache_api_data_news](<https://github.com/DavideNas/DEMO-Flutter-Cache-Tests/tree/main/cache_api_data_news>) : Demo with http for api requests and SQFLite for data caching
3. [cache_http_test](<https://github.com/DavideNas/DEMO-Flutter-Cache-Tests/tree/main/cache_http_test>) : Demo with dio and dio_http_cache to manage data stored into device
4. [cached_images_example](<https://github.com/DavideNas/DEMO-Flutter-Cache-Tests/tree/main/cached_images_example>) : Demo to test caching for images provided by flutter_cache_manager
5. [leaderboard_simulator](<https://github.com/DavideNas/DEMO-Flutter-Cache-Tests/tree/main/leaderboard_simulator>) : Demo with custom backend in nodeJS caching network image into internal Hive DB
